# Clean Code Collection

## Genel Bilgiler
- **Yazar**: Robert C. Martin (Uncle Bob)
- **Yayın Yılı**: Clean Code (2008), Clean Coder (2011)
- **Format**: PDF
- **Durum**: ✅ Tamamlandı

## Özet
Clean Code koleksiyonu, yazılım geliştirme dünyasının en önemli kaynaklarından ikisini içeriyor. Clean Code, temiz ve okunabilir kod yazmanın prensiplerini, tekniklerini ve en iyi uygulamalarını detaylı şekilde ele alıyor. Clean Coder ise profesyonel bir yazılım geliştiricinin davranış kodlarını, etik değerlerini ve kariyerini nasıl yöneteceğini anlatıyor.

- [x] **Anlamlı İsimlendirme**: Değişken, fonksiyon ve sınıf isimlendirme prensipleri

- [x] **Sınıf ve Nesne Tasarımı**: SOLID prensipleri ve clean architecture
- [x] **Refactoring**: Kötü kodu temiz koda dönüştürme teknikleri
- [x] **Profesyonellik**: Yazılımcı olarak sorumluluklar ve etik değerler
- [x] **Zaman Yönetimi**: Tahmin, planlama ve deadline yönetimi

## Notlar
### Önemli Alıntılar
- "Clean code always looks like it was written by someone who cares."
- "The only way to go fast is to go well."
- "First make it work, then make it right, then make it fast."
- "Indeed, the ratio of time spent reading versus writing is well over 10 to 1."

### Kişisel Çıkarımlar
- **Clean Code Bölümü**: Gerçekten çok beğendim. Kod yazma alışkanlıklarımı değiştiren önemli prensiplere sahip. Özellikle fonksiyon tasarımı ve isimlendirme konularında çok değerli bilgiler var.
- **Clean Coder Bölümü**: The Pragmatic Programmer ile benzer konuları işlediği için çok üzerine durmadım. Profesyonellik ve sorumluluk konularında güzel öğütler içeriyor ancak önceki okumalardan tanıdık gelen konular.
- Kod yazarken sürekli "Bu kod başkası tarafından okunabilir mi?" sorusunu sormak gerektiğini hatırlattı.

## Uygulamalar
Clean Code prensiplerini günlük kod yazma pratiğinde uyguluyorum:
- Fonksiyonları mümkün olduğunca küçük tutmaya çalışıyorum
- İsimlendirmeye daha fazla özen gösteriyorum
- Gereksiz yorumlar yerine self-documenting code yazmaya odaklanıyorum
- TDD prensiplerini projelerimde kullanıyorum
